Expand description

Functions to create zenoh entities with 'static lifetime.

This trait contains functions to create zenoh entities like Subscriber, and Queryable with a 'static lifetime. This is useful to move zenoh entities to several threads and tasks.

This trait is implemented for Arc<Session>.

Examples

use zenoh::prelude::r#async::*;

let session = zenoh::open(config::peer()).res().await.unwrap().into_arc();
let subscriber = session.declare_subscriber("key/expression")
    .res()
    .await
    .unwrap();
async_std::task::spawn(async move {
    while let Ok(sample) = subscriber.recv_async().await {
        println!("Received : {:?}", sample);
    }
}).await;

Create a Queryable for the given key expression.

Arguments
  • key_expr - The key expression matching the queries the Queryable will reply to
Examples
use zenoh::prelude::r#async::*;

let session = zenoh::open(config::peer()).res().await.unwrap().into_arc();
let queryable = session.declare_queryable("key/expression")
    .res()
    .await
    .unwrap();
async_std::task::spawn(async move {
    while let Ok(query) = queryable.recv_async().await {
        query.reply(Ok(Sample::try_from(
            "key/expression",
            "value",
        ).unwrap())).res().await.unwrap();
    }
}).await;
